# Booklyft

A WordPress booking system plugin with services, booking form, admin dashboard, and availability check.

## Features
- Services management
- Frontend booking form via shortcode
- Admin booking list
- Email confirmation
- AJAX availability check

## Shortcodes
- [booklyft_services]
- [booklyft_booking_form]

## Install
1. Upload the folder to `/wp-content/plugins/`
2. Activate Booklyft in WordPress
3. Add services in the Booklyft admin menu
4. Place shortcodes on pages
